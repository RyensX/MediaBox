## 🗃️功能

作为**插件容器**提供数据展示视图

插件简单实现[插件API](https://github.com/RyensX/MediaBoxPlugin)的组件解析数据即可得到一个完整APP

如一个看视频的APP、一个看漫画的APP，并且多个插件内容可聚合
[MediaBox示例插件-樱花动漫第三方客户端](https://github.com/RyensX/SakuraAnime2Plugin)
